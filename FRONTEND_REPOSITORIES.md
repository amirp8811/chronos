# Frontend Repositories

The CHRONOS application runtimes have been evicted from the core repository to reduce attack surface and build complexity.

## chronos-web
- Consumes `@chronos/engine-wasm` npm package.
- Built from the `chronos-wasm` target in this repo.
- Repository: [Internal/External link]

## chronos-mobile (Android/iOS)
- Consumes native UniFFI / C-ABI headers.
- Built from the `chronos-lite` or `chronos-core` targets.
- Repository: [Internal/External link]
