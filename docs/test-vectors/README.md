# CHRONOS test vectors

This directory contains stable wire-format examples for external implementations.

Current vectors:

- `crp7_data_packet_v1.json` — minimal CRP7 Data packet with stream ID 1, sequence 2, payload `010203`.

More vectors should be added for CHR7 secure cells, CHS7 handshakes, and RTE7 route packets after the protocol formats are frozen.
